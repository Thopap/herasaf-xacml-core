This Package contains the AttributeFinder. An Attribute Finder is used to resolve references to attributes which aren't in the request. According to the specification, attributes not beeing in the request, have to be handled as if they where in the request.

See: OASIS eXtensible Access Control Markup Langugage (XACML) 2.0, Errata 29 June 2006 pages 78, chapter Attribute Retrieval, for further information.
