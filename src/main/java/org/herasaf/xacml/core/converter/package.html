Contains the converter classes used by JAXB to find the right datatype, function, policyCombiningAlgorithm or ruleCombiningAlgorithm and place that into the unmarshalled request.

See: OASIS eXtensible Access Control Markup Langugage (XACML) 2.0, Errata 29 June 2006 appendix A.2 (data types) page 103, appendix A.3 (functions) page 105, appendix B.10 (combining algorithms) page 132, for further information.
