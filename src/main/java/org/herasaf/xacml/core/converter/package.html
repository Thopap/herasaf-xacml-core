Contains the converter classes used by JAXB to find the right datatype, function, policyCombiningAlgorithm or ruleCombiningAlgorithm and place that into the unmarshalled request.

See: OASIS eXtensible Access Control Markup Langugage (XACML) 2.0, Errata 29. January 2008 appendix A.2 (data types) page 110, appendix A.3 (functions) page 113, appendix B.10 (combining algorithms) page 139, for further information.
